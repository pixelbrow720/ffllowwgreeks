Color discipline
Three earned accents.

Red for short-gamma / dealer pressure, emerald for long-gamma / forced-buy, amber for warn / charm peak. Indigo + violet are decorative-only and live in hero ambient lighting. Everything else is grayscale on near-black.

red
emerald
amber
indigo
Typography
Inter + JetBrains Mono.

Inter for prose, JetBrains Mono for numerics with tnum + ss01 + cv11 on by default so digits align in tables. Display sizes use negative letter- spacing for that premium-tight feel.

5810.42 · −$3.84B · +1.24B
Motion
Restraint over spectacle.

Lamp drift in hero, marquee tape, char-by-char title reveal, tracing-beam scroll progress, cursor-anchored spotlight, live- flicker on numerics. No bouncy ease — trading software wants linear-ish motion with the --ease-emph curve for emphasis.

Where it fits

Inside flowjob.id.

no signup · no auth surface
flowjob.id integration

FlowGreeks is one of three products in flowjob.id, alongside the quant-macro fund track and the main curriculum site (order flow + options flow). The parent site owns user accounts, billing, and add-on activation. From this binary's perspective there's no signup / login / refresh token / tier middleware — just opaque API keys provisioned by the parent site.

The activate flow above lives inside flowjob.id; once the secret is copied the user lands on the dashboard at flowgreeks.flowjob.id. That's the integration story — the mockup just shows what each surface looks like.
