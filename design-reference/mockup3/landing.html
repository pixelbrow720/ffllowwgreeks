01 · Thesis

Dealers don't choose to hedge.
Math chooses for them.

Every option contract on a dealer's book has a delta that drifts as spot moves, time decays, and IV breathes. The dealer can't ignore the drift — they are short volatility, mark-to-market, and sized beyond what discretion can absorb. So they buy futures. They sell futures. Mechanically. Predictably. Knowably.

FlowGreeks reconstructs the dealer's book from public OPRA prints + open interest, computes their forced flow window forward, and publishes it before the move clears the tape.

Predictive forced flow

Dollars, not vibes.

Translate dealer position into the dollar notional they have to execute over the next 15 / 30 / 60 minutes. SpotGamma shows you gamma; we show you the order they're about to send.

Charm Clock

Time is a force.

Five intraday zones — WEAK, RISING, PEAK, FADING, PIN — based on aggregated charm velocity. Tells you when dealer hedging is accelerating, plateauing, or capitulating.

0DTE only

Surface ≠ noise.

We don't watch 3,500 tickers. We watch 2 — SPX and NDX 0DTE, the chains where dealer hedging actually moves index futures. Everything else is decoration.

02 · Pipeline

From wire to signal in under one second.

Five stages, one Postgres, one NATS bus, one binary per role. Built in Go for the latency and the simplicity.

01

Ingest

OPRA pillar + GLBX MDP3 over Databento WebSocket. 90-byte fixed-layout encoder, zero-alloc hot path.

02

Normalize

Vendor records → typed Tick → NATS subjects. ticks.<sym>.<type>

03

Compute

BS pricing 105ns. IV solver 1.03µs (Brent + warm-start). Greeks 259ns. GEX 5.2µs / 200 strikes.

04

Aggregate

1Hz aggregator publishes state.<sym>.gex + archives dealer_state_1s.

05

Distribute

REST snapshots + WebSocket fanout. /ws/live drop-on-slow, 15s heartbeat.

04 · Integrate

One header. One endpoint.

Activated as a flowjob.id add-on. The parent site mints your API key once. Hit any endpoint with that bearer token from any client.

REST · snapshot GET /api/snapshot/spx 
# latest dealer state for SPX 0DTE
curl https://api.flowgreeks.flowjob.id/api/snapshot/spx \
  -H "Authorization: Bearer $FLOWGREEKS_KEY"

# response
{
  "net_gex": -3840000000,
  "regime": 2,
  "dpi": { "composite": 82, "charm": 74 },
  "pin": { "top_strike": 5810, "top_prob": 0.41 }
}
WebSocket · live stream /ws/live 
// subscribe to live state + alerts
const ws = new WebSocket("wss://api.flowgreeks.flowjob.id/ws/live");
ws.onopen = () => ws.send(JSON.stringify({
  action: "subscribe",
  symbols: ["spx", "ndx"],
  kinds: ["gex", "narrative", "alert"],
}));
ws.onmessage = (e) => {
  const evt = JSON.parse(e.data);
  if (evt.type === "snapshot") updateUI(evt);
};
